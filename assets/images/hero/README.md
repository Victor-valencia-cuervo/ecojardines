# 🖼️  Imágenes – Hero (Portada)

Coloca aquí la imagen de fondo de la sección principal.

| Archivo       | Descripción                          | Tamaño recomendado |
|---------------|--------------------------------------|--------------------|
| hero-bg.jpg   | Paisaje verde, jardín o lote cuidado | 1920 × 1080 px     |
