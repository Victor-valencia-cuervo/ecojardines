# 🖼️  Imágenes – Casos (Antes y Después)

Coloca aquí las fotos de transformaciones. Cada caso muestra
una imagen "Antes" y una "Después" lado a lado.

| Archivo         | Etiqueta  | Qué mostrar              | Tamaño recomendado |
|-----------------|-----------|--------------------------|--------------------|
| antes-01.jpg    | ANTES     | Lote con maleza alta     | 400 × 300 px       |
| despues-01.jpg  | DESPUÉS   | Lote limpio y cortado    | 400 × 300 px       |
| antes-02.jpg    | ANTES     | Jardín descuidado / seco | 400 × 300 px       |
| despues-02.jpg  | DESPUÉS   | Jardín verde restaurado  | 400 × 300 px       |

> **Tip:** Intenta que las fotos de Antes y Después sean del mismo ángulo
> y encuadre — el contraste visual es más impactante.
>
> Para agregar más casos, duplica el bloque `<!-- Caso N -->` en el HTML
> y agrega los archivos `antes-03.jpg` / `despues-03.jpg`, etc.
